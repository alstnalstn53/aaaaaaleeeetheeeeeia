# Archive Project Images

각 프로젝트 폴더에 이미지 3장씩 넣으세요.

## 파일 규격
- `main.jpg` — 메인 이미지 (와이드, 21:9 비율 권장)
- `detail-01.jpg` — 디테일 1 (4:3 비율 권장)
- `detail-02.jpg` — 디테일 2 (4:3 비율 권장)

## 폴더 구조
```
images/
├── gyeol/          ← GYEOL (Cafe & Bakery)
│   ├── main.jpg
│   ├── detail-01.jpg
│   └── detail-02.jpg
├── raw-concrete/   ← RAW CONCRETE (Space Design)
├── structure-01/   ← STRUCTURE 01 (Fashion)
├── deep-soil/      ← DEEP SOIL (Fragrance)
├── the-void/       ← THE VOID (Exhibition)
└── monolith/       ← MONOLITH (Architecture)
```

## 참고
- jpg, png, webp 모두 가능
- 이미지가 없으면 패턴 플레이스홀더가 자동으로 표시됨
- GitHub에 직접 업로드 → Vercel이 자동으로 정적 파일 서빙
- 권장 이미지 크기: main 1600x680px, detail 800x600px
